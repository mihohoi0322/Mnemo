# MVPスコープ定義

## 1. 目的
- 個人のメモ整理を目的に、スクリーンショットをAIで自動タグ付けし、自然文検索で見つけられる体験を提供する。

## 2. MVPで「やること」
- iOSアプリでスクリーンショットを取り込める
- ローカル保存（画像・メタデータ）
- クラウド解析
  - OCR
  - 自動タグ生成
  - 画像埋め込み（意味検索）
- 自然文検索（意味検索）
- タグ編集（自動/手動の区別）
- コレクション作成・追加
- ライブラリでの一括操作（タグ/コレクション/削除）
- 解析待ち/失敗の表示と再試行
- 完全削除（確認ダイアログ必須）

## 3. MVPで「やらないこと」
- クラウド同期
- Web版の閲覧・編集
- 共有機能
- 多端末同期
- 高度なコレクション機能（ネスト/スマートコレクション）
- 自動タグの学習・個人最適化
- 重い分析（画像分類の詳細レポート）

## 4. 成功条件（MVPの完了基準）
- ユーザーが「自然文検索で目的の画像を見つけられる」体験が成立する
- タグ編集・コレクション整理が最低限のストレスで行える
- 解析失敗時に再試行でき、アプリが破綻しない

## 5. 依存関係 / 前提
- 解析サービスが安定稼働していること
- iOSでの画像アクセス権限が取得できること
- ローカルDBと検索インデックスが端末で利用可能であること
