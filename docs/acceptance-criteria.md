# 受け入れ基準（MVPチェックリスト）

## 1. 取り込み
- スクリーンショットを取り込める
- 取り込み後、ローカル保存される

## 2. 解析
- 画像がクラウド解析に送信される
- OCR結果が取得できる
- 自動タグが付与される
- 解析失敗時に再試行できる

## 3. 検索
- 自然文検索で結果が返る
- 検索結果が関連度順で並ぶ
- OCRテキストも検索対象になる

## 4. 整理
- タグを手動で追加/削除できる
- コレクションを作成・追加できる
- ライブラリで複数選択して一括操作できる

## 5. 削除
- 削除時に確認ダイアログが出る
- 削除は完全削除で復元できない

## 6. 非機能
- 1000件程度でも検索が1秒以内
- オフライン時は解析をキュー保持する
- プライバシー設定（クラウド解析ON/OFF）が機能する
