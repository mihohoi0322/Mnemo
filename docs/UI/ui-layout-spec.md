# UIレイアウト仕様書（Figma貼り付け用）

## 0. 前提
- プラットフォーム: iOS（MVP）。
- 画面サイズ: iPhone 14 Pro（393 x 852）を基準。
- レイアウト方式: 8ptグリッド + 4pt単位。

## 1. 共通ルール
- 余白
  - 画面左右: 16
  - セクション上下: 16〜24
  - セクション内要素間: 8〜12
- タイポ
  - 画面タイトル: 24/32 SemiBold
  - セクション見出し: 16/22 Medium
  - 本文: 14/20 Regular
  - 補助: 12/16 Regular
- 角丸
  - カード: 12
  - チップ/ボタン: 16
- 影
  - 1段階の浅い影（カードのみ）

## 2. コンポーネント（先に作ると早い）
- 検索バー
  - 高さ 44、左右アイコン + プレースホルダ
- タグチップ
  - 高さ 28、左右 12、丸角 16
  - 自動タグ: 薄い塗り
  - 手動タグ: 濃い塗り
- 画像サムネカード
  - 1:1 正方形
  - 角丸 12、影あり
- セクションヘッダー
  - 左: タイトル、右: "See all"
- フィルタチップ
  - 高さ 32、選択状態で塗り
- 空状態コンポーネント
  - アイコン + 短文 + CTA

## 3. 画面仕様

### 3.1 検索（ホーム）
- Frame: 393x852
- 構成
  1. ヘッダー（上部）
     - 左: タイトル「Mnemo」
     - 右: 設定アイコン
  2. 検索バー
  3. サジェスト（横スクロール）
     - 最近検索 / よく使うタグ / コレクション
  4. 最近見た（横スクロールカード）
  5. 最近取り込んだ（横スクロールカード）
  6. 解析待ち/失敗（小さなバナー）

### 3.2 検索結果
- 構成
  1. 上部: 検索バー（戻る付き）
  2. フィルタ行（タグ/コレクション/期間）
  3. 並び替え（右上ドロップダウン）
  4. 結果グリッド（2列）
  5. 複数選択（下部ツールバー）

### 3.3 詳細
- 構成
  1. 画像ビュー（上部、全幅）
  2. クイック操作（タグ編集 / コレクション追加 / 削除）
  3. タグ（手動/自動の区別）
  4. OCRテキスト（折りたたみ）
  5. 関連候補（横スクロール）
- 削除
  - 右上 or 画面下部に配置。
  - 必ず確認ダイアログ。

### 3.4 タグ編集（モーダル）
- 構成
  1. タイトル「タグ編集」
  2. タグ追加入力 + サジェスト
  3. 現在タグ一覧（削除）
  4. 保存 / キャンセル

### 3.5 ライブラリ
- 構成
  1. 上部: タイトル + フィルタ
  2. 全件グリッド（2列）
  3. 複数選択モード
  4. 一括操作（タグ付与/削除・コレクション追加・削除）

### 3.6 コレクション
- 構成
  1. 一覧（カード or リスト）
  2. 作成ボタン
  3. コレクション詳細（中身グリッド）

### 3.7 解析キュー
- 構成
  1. 状態別セクション（待ち/中/失敗）
  2. 失敗は再試行CTA

### 3.8 設定
- 構成
  1. クラウド解析 ON/OFF + 説明
  2. 通信条件（Wi-Fiのみ）
  3. 将来: バックアップ/エクスポート

## 4. 画面作成順（おすすめ）
1. 検索（ホーム）
2. 検索結果
3. 詳細
4. ライブラリ
5. コレクション
6. 解析キュー
7. 設定
