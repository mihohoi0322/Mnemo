# UIラフ（スタイル適用版 / 温かいトーン）

## 1. 検索（ホーム）
- 背景: #F7F4EF
- ヘッダー: タイトル「Mnemo」(24/32 SemiBold, #1E1E1E)
- 検索バー: 白背景 + 枠線 #E6E0D7
- サジェスト: チップ（自動=淡緑, 手動=淡青）
- 最近見た/最近取り込んだ: サムネカード（角丸12 + 影）
- 解析バナー: 薄い警告色（#C56A5B）

## 2. 検索結果
- 背景: #F7F4EF
- 検索バー: 常時表示（戻る付き）
- フィルタチップ: 選択時は Primary (#2F5D50)
- 結果グリッド: 2列、カード影あり
- 複数選択バー: Surface + Primaryボタン

## 3. 詳細
- 画像ビュー: 白背景に載せる
- クイック操作: アイコン + テキスト、Primary色
- タグ: 自動=淡緑、手動=淡青 + 太字
- OCR: 折りたたみでセクション表示
- 関連候補: 横スクロールカード
- 削除: Warning色で強調

## 4. タグ編集（モーダル）
- 背景: 半透明オーバーレイ
- モーダル: 白 + 角丸16
- ボタン: Save=Primary、Cancel=Secondary

## 5. ライブラリ
- ヘッダー + フィルタ行
- グリッド: 2列
- 一括操作バー: 固定
- 削除はWarning色

## 6. コレクション
- 一覧はカード型
- 作成ボタンはAccent (#D88C5A)
- 詳細はグリッド表示

## 7. 解析キュー
- 状態ごとに薄い区切り背景
- 失敗はWarning色 + 再試行ボタン

## 8. 設定
- セル型リスト
- ON/OFFスイッチはPrimary色
- 説明文はText Secondary
