# UIコンポーネント一覧（Figma用）

## 1. 基本コンポーネント

### 1.1 ボタン
- Primary
  - 高さ: 44
  - 角丸: 16
  - 背景: #2F5D50
  - 文字: #FFFFFF (14/20 Medium)
- Secondary
  - 高さ: 44
  - 角丸: 16
  - 背景: #FFFFFF
  - 枠線: #E6E0D7
  - 文字: #2F5D50
- Destructive
  - 高さ: 44
  - 角丸: 16
  - 背景: #C56A5B
  - 文字: #FFFFFF

### 1.2 アイコンボタン
- サイズ: 40
- 角丸: 12
- 背景: #FFFFFF
- 枠線: #E6E0D7

### 1.3 検索バー
- 高さ: 44
- 角丸: 12
- 背景: #FFFFFF
- 枠線: #E6E0D7
- 左: 検索アイコン
- プレースホルダ: 14/20 Regular #6B6B6B

### 1.4 タグチップ
- 高さ: 28
- 角丸: 16
- 余白: 左右 12 / 上下 6
- 自動タグ
  - 背景: #E7EFEA
  - 文字: #2F5D50 (12/16 Regular)
- 手動タグ
  - 背景: #DDE8F2
  - 文字: #2F5D50 (12/16 SemiBold)

### 1.5 フィルタチップ
- 高さ: 32
- 角丸: 16
- 余白: 左右 12
- 通常: 背景 #FFFFFF / 枠線 #E6E0D7
- 選択: 背景 #2F5D50 / 文字 #FFFFFF

### 1.6 セクションヘッダー
- 左: タイトル 16/22 Medium
- 右: "See all" 12/16 Regular #6B6B6B

### 1.7 カード（サムネ）
- サイズ: 160 x 160
- 角丸: 12
- 影: 0 2 8 rgba(0,0,0,0.08)
- キャプション（任意）: 12/16 #6B6B6B

### 1.8 バナー（解析状態）
- 高さ: 40
- 角丸: 12
- 背景: #F3E2DF
- 文字: #C56A5B 12/16

### 1.9 トグル
- iOS標準トグルを想定
- ON時: #2F5D50

## 2. 複合コンポーネント

### 2.1 画像グリッド（2列）
- 列間: 12
- 行間: 12
- 左右余白: 16

### 2.2 検索結果ツールバー
- 高さ: 56
- 背景: #FFFFFF
- 左: 選択数
- 右: アクション（Add Tag / Add Collection / Delete）

### 2.3 OCRテキストブロック
- タイトル: "OCR"
- 折りたたみアイコン
- 本文: 14/20 Regular

### 2.4 コレクションカード
- 高さ: 72
- 角丸: 12
- 背景: #FFFFFF
- タイトル: 14/20 Medium
- 補助: 12/16 #6B6B6B

### 2.5 空状態
- アイコン + テキスト + CTA
- 例: "まだ画像がありません" / "スクショを保存してみよう"
