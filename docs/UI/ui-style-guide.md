# UIスタイルガイド（初期案 / 温かいトーン）

## 1. コンセプト
- ノート/紙の質感を感じる、柔らかく落ち着いた整理体験。
- 文字の可読性と検索の即時性を優先しつつ、温度感のある配色。

## 2. カラー
- Background: #F7F4EF
- Surface: #FFFFFF
- Primary: #2F5D50
- Primary Light: #5C8B7D
- Accent: #D88C5A
- Text Primary: #1E1E1E
- Text Secondary: #6B6B6B
- Border/Subtle: #E6E0D7
- Tag Auto BG: #E7EFEA
- Tag Manual BG: #DDE8F2
- Warning: #C56A5B

## 3. タイポ
- 見出し: 24/32 SemiBold
- セクション: 16/22 Medium
- 本文: 14/20 Regular
- 補助: 12/16 Regular

## 4. 角丸・影
- 角丸
  - カード: 12
  - チップ/ボタン: 16
  - 検索バー: 12
- 影
  - カード: 0 2 8 rgba(0,0,0,0.08)

## 5. コンポーネントの見た目
- 検索バー
  - 背景: Surface
  - 枠線: Border/Subtle
  - 角丸: 12
- タグ
  - 自動タグ: Tag Auto BG + Primary
  - 手動タグ: Tag Manual BG + Primary
  - 手動タグは太字で視認性を上げる
- サムネカード
  - 角丸 12 + 影
  - タイトル/補助テキストは下に小さく

## 6. 余白
- 左右: 16
- セクション間: 16〜24
- 要素間: 8〜12

## 7. アイコン
- 線の細いシンプルアイコン
- 角の丸いスタイルを優先

## 8. 空状態
- 小さなイラスト + 一文 + 行動CTA
- 例: "まだ画像がありません" + "スクショを保存してみよう"
